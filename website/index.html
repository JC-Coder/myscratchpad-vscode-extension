MyScratchPad is a lightweight extension for VS Code-based editors. It gives every temporary thought a home so you stop juggling sticky notes, random files, and lost clipboard content.

Works anywhere VS Code extensions run, including VS Code, Cursor, Windsurf, Trae, and Antigravity.

Why This Project Exists

Developers create dozens of temporary files daily. Most get lost.

MyScratchPad keeps throwaway work intentional: quick notes, debug logs, query drafts, and snippets are all easy to create and easy to clean up inside your editor.

Step 1

Open command palette

`MyScratchPad: New`

Step 2

Pick file extension

`.md`, `.ts`, `.json`, `.sh`

Step 3

Write and move on
